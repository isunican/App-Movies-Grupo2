📌 Buscar contenido por nombre

⬅️ Volver al inicio

Descripción

Como usuario, quiero tener acceso a una barra de búsqueda con el objetivo de escribir nombres de las series y la lista de series cambie con base a lo que escribo

Funcionamiento

  1. Abrir la aplicación.
  2. Pulsar en la barra de búsqueda
  3. Escribir el nombre del contenido que se desea buscar.
  4. La lista de series se actualizará automáticamente para mostrar solo aquellas que coincidan con el término de búsqueda ingresado.
