📌 Ver información detallada de una serie

⬅️ Volver al inicio

Descripción

Como usuario, quiero poder ver la información detallada de una serie para conocer sus detalles como género, año, capítulos, puntuaciones, etc.

Funcionamiento

  1. Abrir la aplicación.
  2. Pulsar en cualquier película.
  3. Podemos ver ya toda la inforación detallada.
